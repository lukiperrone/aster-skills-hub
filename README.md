## 📋 About aster-skills-hub

aster-skills-hub is a set of agent skills that help you use the Aster Futures API. It lets you deposit funds from your wallet and use both versions of the API: v1 (HMAC) and v3 (EIP-712). You can connect to the market, check your account balances, place and manage orders, and follow live market streams easily. It also takes care of common issues like errors and rate limits for you.

This project is useful if you want to interact with Aster Futures through ready-made tools without writing complex code yourself.
